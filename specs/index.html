SLED Specifications for common processors

These specifications describe the binary and assembly-language representations of instructions used on several popular processors. From a specification like this, the New Jersey Machine-Code Toolkit can generate code to emit or recognize (assemble or disassemble) these instructions.

The specifications themselves have been converted automatically from TeX files, so they may look strange in spots---especially around opcode tables. We have complete specifications for these machines:

  • Mips,
  • Sparc,
  • Pentium, and
  • Alpha.
